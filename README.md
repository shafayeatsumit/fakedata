# Usage
*	install `fake`

``` 
pip install fake-factory
```

*	install `pandas`

[pandas 安裝指示](http://pandas.pydata.org/pandas-docs/stable/install.html)

*	move to the directory
*	run 

``` python
python src/fake.py
```

*	see data file you just generate in `./datas`!!
